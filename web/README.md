# vomowa-web

## Overview

Next.js web application for vomowa - the Transparency Card dashboard.

## Tech Stack

- Next.js 15 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- Recharts (charts)

## Features

- 📊 Statistics Dashboard
- 👤 Politician Profiles
- 🔔 Notification Settings
- 📱 Mobile-First Design
- 🌙 Dark Mode

## Installation

```bash
cd web
npm install
npm run dev
```

## Environment Variables

```env
NEXT_PUBLIC_API_URL=http://localhost:3001
```
